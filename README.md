My Dotfiles for OS X
===========

## Installation

Execute the following in a terminal

```bash
git clone https://github.com/kernel-panic96/dotfiles-mac.git ~/dotfiles
cd ~/dotfiles
chmod +x bootstrap.sh && ./bootstrap.sh
```
